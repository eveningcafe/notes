---
title: Escape characters
---

| Code   | Result                                                           |
| ------ | ---------------------------------------------------------------- |
| `\'`   | Single quote                                                     |
| `\\`   | Backslash                                                        |
| `\n`   | [New line](https://en.wikipedia.org/wiki/Newline)                |
| `\r`   | [Carriage return](https://en.wikipedia.org/wiki/Carriage_return) |
| `\t`   | Tab                                                              |
| `\b`   | Backspace                                                        |
| `\f`   | Form Feed                                                        |
| `\ooo` | Octal value                                                      |
| `\xhh` | Hex value                                                        |
